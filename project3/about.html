Ship Shooter

High Concept

Ship Shooter is a game where you need to defeat the oncoming waves of enemy pirate ships.
You can upgrade your ship and gain new abilities with points you've won in previous rounds.

Genre

- Action
- Arcade

Platform

Desktop web

Story

You're a pirate trying to fight off the enemy pirates.

Aesthetics

- Pixel Art
- 8 Bit

Gameplay

The user will go through waves of enemies, as the user progresses the harder it becomes.
The user will be able to make upgrades to their ship using the points they won the rounds prior
