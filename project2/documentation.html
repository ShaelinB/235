Project 2 : Pokemon Guide

API : PokeAPI

    Reference JSON links for Pikachu:
  • JSON that contains all the pokemon in a specific generation
  • JSON that contains the url to the evolution chain for that pokemon
  • JSON that contains the evolution chain for that pokemon
  • JSON that contains the stats for that pokemon

Resources:
Color Pallete
Interface Sketch

Process of gathering information:
1. The user enters a pokemon and the generation they're in and selects what type of info they want to recieve
2. Once the user hits select it makes a link using the generation.
3. That link is used to check if that pokemon is in that generation. If it isn't it displays an error, otherwise it retrieves the link for the species

If you chose stats:
4. Once you check if that pokemon exists it creates a new link specific to the pokemon
5. This link contains the data for it's type, base stats, and image.
6. Displays data

If you chose evolution:
4. Using that link you can get to the evolution chain for that species.
5. Displays data using a recursive method to go through the whole evolution tree.

3 UI Controls:
search bar
generation dropdown
information dropdown

Additional Notes:
1. The status changes to searching for pokemonName once the button is clicked but the data is retrieved really fast so you can't see it
2. CSS uses flexbox
3. The website is responsive
